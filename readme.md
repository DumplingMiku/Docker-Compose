# Install Docker
`curl -fsSL https://get.docker.com/ | sudo sh`
# Create/Update Docker Compose
`docker compose up -d`
# Remove Docker Compose
`docker compose down`
# Start Docker Compose
`docker compose start`
# Stop Docker Compose
`docker compose stop`
